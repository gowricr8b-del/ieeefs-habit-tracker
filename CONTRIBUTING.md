# Contributing to Habit Tracker

Thank you for your interest in contributing! This project is designed for Patchfest participants and welcomes contributions from all skill levels.

## How to Contribute

1. Fork the repo
2. Clone your fork
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit with clear messages: `git commit -m "feat: add habit streak calculation"`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Open a Pull Request

---

## Issues

- Check the **Issues** tab for available tasks.
- Issues are labeled with difficulty: `easy`, `medium`, `hard`.
- Assign yourself to an issue before starting.

---

## Code Style

- Use 2-space indentation
- Use `camelCase` for variables and functions
- Functional components preferred in React
- Tailwind CSS for styling

---

## Pull Requests

- Describe your change clearly
- Reference the issue number in your PR
- Make sure frontend and backend are both working
- Test your changes locally

---

## Communication

- For questions or clarifications, leave a comment on the issue or PR
- Be respectful and provide constructive feedback
